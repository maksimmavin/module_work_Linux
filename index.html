GNU/Linux: Основи та Використання

Проєкт створив: Король Максим

1. Коротка інформація про ОС GNU/Linux

GNU/Linux — це операційна система з відкритим кодом, яка базується на ядрі Linux, розробленому Лінусом Торвальдсом у 1991 році. Вона відзначається високою стабільністю, безпекою та налаштовуваністю, що робить її популярною серед розробників і серверних систем.

3. Приклади команд роботи в Linux CLI

  • pwd — виводить поточний каталог
  • echo — виводить текст або змінні
  • ls — показує вміст каталогу
  • cd — змінює поточний каталог
  • touch — створює новий файл
  • mkdir — створює нову директорію
  • cp — копіює файли або каталоги
  • mv — переміщує або перейменовує файли
  • rm — видаляє файли або директорії
  • history — показує історію виконаних команд

4. BASH-cкрипт, що імітує авторизацію в системі

Простий BASH-сценарій, який запитує ім'я користувача та пароль:


#!/bin/bash
echo "Введіть ім'я користувача:"
read username
echo "Введіть пароль:"
read -s password

if [[ $username == "admin" && $password == "admin" ]]; then
    echo "Авторизація успішна!"
else
    echo "Невірне ім'я користувача або пароль!"
fi
